# 🎮 Simon Says Game

A sleek, interactive memory game built using **Vanilla JavaScript**, **HTML5**, and **CSS3**. 

## 🎯 How to Play
1. **Start:** Press any key on your keyboard to begin the game.
2. **Memorize:** Watch the sequence of colors as they flash on the screen.
3. **Repeat:** Click the buttons in the exact same order.
4. **Progress:** Each successful round adds one more flash to the sequence.
5. **Game Over:** One wrong click ends the game and resets your progress.

## ✨ Key Features
*   **High Score Tracking:** Keeps track of your best performance during the session.
*   **Visual Feedback:** Custom CSS animations for "Game Flash" and "User Click" states.
*   **Responsive Logic:** Written with optimized event listeners and DOM manipulation.
*   **Game Over Alert:** The background flashes red briefly to signal an incorrect move.

## 🛠️ Technical Stack
*   **Frontend:** HTML5, CSS3
*   **Logic:** JavaScript (DOM, Promises, Event Loops)

## 📂 Project Structure
* `index.html` - The layout and structure.
* `style.css` - Custom styling and animations.
* `app.js` - The core game engine and sequence logic.
